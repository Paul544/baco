RFS Baco Help

Baco support application can help Baco Support Operators with monitoring of crews.

This tool is an aid not a replacement for the BACO board but can remove some of the complexity of monitoring current cylinder pressure and working times.

Key Features:

  • Supports many teams
  • Different Cylinders sizes
  • Record names
  • Calculate end time / remainming minutes
  • Alerts user when crew reaches 10 minutes of remaining time
  • Works offline
